# Cotización del Dólar

> Sistema que extrae del sitio web del MSPBS la cantidad de personas confirmadas con COVID19.

### Como se instala ?

```bash
$ git clone https://github.com/cabupy/covid19-py.git
$ cd covid19-py
$ npm install
```
